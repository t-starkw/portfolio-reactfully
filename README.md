# React Portfolio
>### Module 20 UNC Coding Bootcamp Challenge - React Portfolio <br><br>
> This application is a single-page react portfolio intended to showcase my most recent work as a developer, while also allowing me to explore my newly acquired React skills. Upon loading the portfolio, users will find an About Me landing page and navigation bar, which routes to sections for my work, my resume, and a contact form. Additionally, the bottom of the page features a footer with icons which link to my various social media contacts. <br><br>

## Installation

Clone the repository:

```sh
git clone git@github.com:t-starkw/portfolio-reactfully.git
```

Install the required dependencies:

```sh
npm install
```

Run the project in your local browser:

```sh
npm start
```
